# Food Retention Analysis

## Overview
This project focuses on food retention and customer churn deep dives. It aims to generate retention data, build unified master datasets, and analyze churn in food delivery or relevant businesses.

## Structure
- `data/`: Datasets for retention analysis.
- `docs/`: Documentation and notes.
- `notebooks/`:
  - `feature_discovery_and_churn.ipynb`: Exploring features indicating retention or churn.
  - `leaky_bucket_deepdive.ipynb`: Analyzing drop-offs and leaky bucket problems.
- `scripts/`: Generation and master building scripts.
  - `generate_retention_data.py`
  - `util_build_unified_master.py`

## Setup
Run the scripts in the `scripts/` directory to build the unified master data before executing the notebooks.
